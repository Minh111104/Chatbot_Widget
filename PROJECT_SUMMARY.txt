STEP 1: Deploy to Vercel
─────────────────────────
1. Visit: https://vercel.com/new
2. Import: Minh111104/Chatbot_Widget
3. Add environment variable:
   Key:   OPENAI_API_KEY
   Value: sk-your-openai-api-key
4. Click: Deploy
5. Copy deployment URL

STEP 2: Integrate with GitHub Pages
────────────────────────────────────
1. Go to: https://github.com/Minh111104/minh111104.github.io
2. Edit: index.html
3. Add before </body>:

   <iframe 
     src="https://YOUR-DEPLOYMENT-URL.vercel.app" 
     style="position: fixed; bottom: 0; right: 0; 
            width: 420px; height: 650px; border: none; 
            z-index: 9999;">
   </iframe>

4. Commit and push
5. Visit: https://minh111104.github.io/
6. Chatbot is live! 🎉
