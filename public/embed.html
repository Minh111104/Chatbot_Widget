ü§ñ Chatbot Widget - Embedding Instructions

Option 1: iFrame Embed (Recommended for GitHub Pages)

Add this code to your website (e.g., https://minh111104.github.io/):

<!-- Add before closing </body> tag -->
<iframe 
  id="chatbot-widget"
  src="https://your-chatbot-domain.vercel.app" 
  style="position: fixed; bottom: 0; right: 0; width: 420px; height: 650px; border: none; z-index: 9999; pointer-events: none;"
  allow="clipboard-write"
></iframe>

<script>
  // Enable pointer events only on the widget area
  const iframe = document.getElementById('chatbot-widget');
  iframe.style.pointerEvents = 'auto';
</script>

Option 2: Standalone Widget (More Control)

If you want the widget button to blend with your main site:

<!-- Add to your <head> -->
<link rel="stylesheet" href="https://your-chatbot-domain.vercel.app/widget.css">

<!-- Add before closing </body> tag -->
<div id="chatbot-root"></div>
<script src="https://your-chatbot-domain.vercel.app/widget.js"></script>

Option 3: Direct Component Integration

If your main website also uses React/Next.js:

import ChatWidget from '@/components/ChatWidget'

function YourPage() {
  return (
    <div>
      {/* Your page content */}
      <ChatWidget />
    </div>
  )
}

üìã Deployment Steps

  1. Deploy to Vercel:
    • Push this repository to GitHub
    • Go to vercel.com and import your repository
    • Add your OPENAI_API_KEY in environment variables
    • Deploy!
  2. Get Your Deployment URL:
    • After deployment, you'll get a URL like chatbot-widget-xyz.vercel.app
  3. Add to Your GitHub Pages Site:
    • Go to your main website repository (minh111104.github.io)
    • Edit your index.html file
    • Add the iframe embed code before </body>
    • Commit and push - the chatbot will appear on your site!
